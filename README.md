# 🌟 Overview

**PostForge AI** is a modern AI-powered LinkedIn content creation platform designed to help professionals establish a stronger personal brand.

Instead of spending hours writing posts, users can generate engaging, professional, and story-driven LinkedIn content within seconds using Google's latest AI models.

Whether you're a student, software engineer, freelancer, founder, creator, or job seeker, PostForge AI helps you create content that builds credibility and increases engagement.

# 💻 Local Development

## Prerequisites

- Node.js 18+
- npm
- Supabase Project
- Google AI Studio API Key

---

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/postforge-ai.git
```

Navigate into the project

```bash
cd postforge-ai
```

Install dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env.local`

```env
NEXT_PUBLIC_SITE_URL=http://localhost:3000

NEXT_PUBLIC_SUPABASE_URL=

NEXT_PUBLIC_SUPABASE_ANON_KEY=

SUPABASE_SERVICE_ROLE_KEY=

GOOGLE_API_KEY=

AI_PROVIDER=google

AI_MODEL=gemini-2.5-flash

POST_GENERATOR_MODEL=gemini-2.5-flash

CONTENT_IMPROVER_MODEL=gemini-2.5-flash

ACHIEVEMENT_MODEL=gemini-2.5-flash

CASE_STUDY_MODEL=gemini-2.5-flash

RESUME_MODEL=gemini-2.5-flash

IMAGE_MODEL=gemini-2.5-flash
```

---

## Start Development Server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
